# iVolve Monthly Tasks Repository

## Overview
This repository contains all the monthly tasks and labs completed as part of the **training program with iVolve** under the supervision of **NTI (National Telecommunication Institute)**.  

The tasks cover practical exercises and hands-on experience in modern DevOps and cloud technologies, including containerization, automation, CI/CD, and orchestration.

---

## Monthly Tasks

The monthly tasks completed include:

1. **Build Tools Overview (Maven & Gradle)**  
   - Learning dependency management, project builds, and automation using Maven and Gradle.

2. **Containerization with Docker**  
   - Lab 3: Run Java Spring Boot App in a Container 
   - Lab 4: Run Java Spring Boot App in a Container
   - Lab 5: Multi-Stage Build for a Node.js App
   - Lab 6: Managing Docker Environment Variables Across Build and Runtime
   - Lab 7: Docker Volume and Bind Mount with Nginx
